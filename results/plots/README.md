## Как читать графики предсказаний

- Каждая точка — один образец бетона из тестовой выборки (206 образцов)
- **Ось X** — реальная прочность (измеренная в лаборатории)
- **Ось Y** — прочность, предсказанная моделью
- **Красная пунктирная линия** — идеальный прогноз (когда предсказание = реальность)
- Чем плотнее точки вдоль красной линии — тем лучше модель
