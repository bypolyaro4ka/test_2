# Папка data — исходные и подготовленные данные

Здесь хранятся данные, которые используются всеми моделями проекта.

---

## Файлы

### `dataset.csv`
Полный датасет Concrete Compressive Strength в формате CSV (1030 записей).

Содержит 9 столбцов:
| Столбец | Описание | Единицы |
|---|---|---|
| Цемент (кг/м³) | Количество цемента в смеси | кг/м³ |
| Шлак (кг/м³) | Доменный шлак | кг/м³ |
| Зола (кг/м³) | Зола-унос (продукт сгорания угля) | кг/м³ |
| Вода (кг/м³) | Количество воды | кг/м³ |
| Добавка (кг/м³) | Суперпластификатор | кг/м³ |
| Крупный щебень (кг/м³) | Крупный заполнитель | кг/м³ |
| Мелкий щебень (кг/м³) | Мелкий заполнитель (менее 4.75 мм) | кг/м³ |
| Возраст (дни) | Возраст бетона на момент испытания | дни (1–365) |
| Прочность бетона (МПа) | Прочность на сжатие (целевая переменная) | МПа |

Используется дашбордом (`dashboard.py`) для отображения таблиц, корреляций и распределений.

### `prepared_data.pkl`
Подготовленные данные в формате pickle. Создаётся скриптом `data_preprocessing.py` и загружается всеми скриптами моделей через `utils.load_data()`.

Содержит словарь со следующими ключами:
| Ключ | Тип | Описание |
|---|---|---|
| `X_train` | numpy array (824, 8) | Обучающая выборка (масштабированная) |
| `X_test` | numpy array (206, 8) | Тестовая выборка (масштабированная) |
| `y_train` | numpy array (824,) | Реальные значения прочности (обучение) |
| `y_test` | numpy array (206,) | Реальные значения прочности (тест) |
| `feature_names` | list | Названия 8 призна��ов |
| `scaler` | StandardScaler | Объект масштабирования (для обратного преобразования) |

Данные разделены в соотношении 80/20 (обучение/тест) с `random_state=42` для воспроизводимости. Все признаки масштабированы с помощью `StandardScaler` (среднее = 0, стандартное отклонение = 1).
