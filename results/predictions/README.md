# Папка predictions — предсказания моделей

Здесь хранятся массивы NumPy (.npy) с предсказанными значениями прочности бетона для каждой модели. Каждый файл содержит 206 чисел — предсказания на тестовой выборке.

---

## Файлы

| Файл | Модель | Тип |
|---|---|---|
| `linear_regression.npy` | Линейная регрессия | Классический |
| `lasso.npy` | Лассо-регрессия | Классический |
| `ridge.npy` | Гребневая регрессия | Классический |
| `random_forest.npy` | Random Forest | Классический |
| `nn_simple.npy` | Линейная регрессия NN | Нейронная сеть |
| `nn_regularized.npy` | Лассо/гребневая регрессия NN | Нейронная сеть |
| `nn_deep.npy` | Random Forest NN | Нейронная сеть |

## Формат данных

Каждый файл — это одномерный массив NumPy из 206 значений типа float. Значения — предсказанная прочность бетона в МПа для каждой записи тестовой выборки.

```python
import numpy as np
preds = np.load('random_forest.npy')
print(preds.shape)  # (206,)
print(preds[:5])    # [38.12, 52.47, 29.85, 44.03, 17.91]
```

## Зачем нужны

Дашборд (`dashboard.py`) загружает эти файлы для:

1. **Графики "предсказанные vs реальные"** — точки рассеяния, где по оси X реальная прочность, а по Y предсказанная. Чем ближе точки к диагонали — тем лучше модель
2. **Таблица проверки на соответствие** — сравнение реальных и предсказанных значений для первых 20 записей с вычислением отклонения
3. **Радарная диаграмма** — для расчета нормализованных метрик сравнения

Благодаря сохранению предсказаний в файлы, дашборд работает мгновенно — не нужно заново обучать модели при каждом открытии.
