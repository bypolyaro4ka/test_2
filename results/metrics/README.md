# Папка metrics — метрики качества моделей

Здесь хранятся результаты оценки каждой модели в формате JSON. Файлы создаются автоматически при запуске скриптов моделей и загружаются дашбордом для отображения сравнительных таблиц и графиков.

---

## Файлы метрик моделей

Каждый файл содержит метрики качества и информацию о модели.

| Файл | Модель | Тип |
|---|---|---|
| `linear_regression.json` | Линейная регрессия | Классический |
| `lasso.json` | Лассо-регрессия | Классический |
| `ridge.json` | Гребневая регрессия | Классический |
| `random_forest.json` | Random Forest | Классический |
| `nn_simple.json` | Линейная регрессия NN | Нейронная сеть |
| `nn_regularized.json` | Лассо/гребневая регрессия NN | Нейронная сеть |
| `nn_deep.json` | Random Forest NN | Нейронная сеть |

### Структура JSON-файла (пример `nn_deep.json`)
```json
{
  "model": "nn_deep",
  "MSE": 37.75,
  "RMSE": 6.14,
  "MAE": 4.41,
  "R2": 0.8535,
  "display_name": "Random Forest NN",
  "type": "neural_network",
  "replaces": "Random Forest Regressor",
  "architecture": "Input(8) -> Dense(256, ReLU) -> Dropout(0.2) -> ...",
  "epochs": 200,
  "params": 43521
}
```

### Описание метрик
| Метрика | Название | Что показывает | Идеальное значение |
|---|---|---|---|
| R2 | Коэффициент детерминации | Доля дисперсии, объясненная моделью | 1.0 |
| MSE | Среднеквадратичная ошибка | Среднее квадратов отклонений | 0 |
| RMSE | Корень MSE | Ошибка в единицах МПа | 0 |
| MAE | Средняя абсолютная ошибка | Среднее модулей отклонений | 0 |

---

## Файлы истории обучения нейронных сетей

Содержат значения потерь (loss) и MAE на каждой эпохе обучения. Используются дашбордом для построения графиков динамики обучения.

| Файл | Модель |
|---|---|
| `nn_simple_history.json` | Линейная регрессия NN |
| `nn_regularized_history.json` | Лассо/гребневая регрессия NN |
| `nn_deep_history.json` | Random Forest NN |

### Структура JSON-файла истории
```json
{
  "loss": [150.2, 98.3, 72.1, ...],
  "mae": [9.1, 7.2, 6.1, ...],
  "val_loss": [140.5, 95.1, 70.8, ...],
  "val_mae": [8.8, 7.0, 5.9, ...]
}
```

| Ключ | Описание |
|---|---|
| `loss` | Потери (MSE) на обучающей выборке по эпохам |
| `mae` | Средняя абсолютная ошибка на обучающей выборке |
| `val_loss` | Потери (MSE) на валидационной выборке по эпохам |
| `val_mae` | Средняя абсолютная ошибка на валидационной выборке |

Каждый массив содержит 200 значений (по одному на эпоху).
